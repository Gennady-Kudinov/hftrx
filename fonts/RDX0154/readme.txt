Находящиеся в этом каталоге файлы - файлы документов программы SG Bitmap Font Editor (дистрибутив в каталоге tools).

bitmap7_bigfont.sgfnt	- большие символы в отображении частоты для SW2011RDX
bitmap7_halffont.sgfnt	- средние ссимволы в отображении частоты для SW2011RDX

chargen_bigfont.sgfnt	- большие символы в отображении частоты для разработок UA1ARN
chargen_halffont.sgfnt	- средние символы в отображении частоты для разработок UA1ARN

chargen_smallfont.sgfnt	- маленькие символы во всех остальных надписях
