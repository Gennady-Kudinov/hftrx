Storch_HF_TRX_CAT.inf
установочный файл ("драйвер") последовательного USB порта для windows XP, Windows7..8.1. Windows 10 уже не требует использования этого файла.

convert_rbf_to_c.cmd
Командный файл преобразования файлов двоисного образа инициализации FPGA в текст на C. Использует windows 32bit утилиту rbf2c.exe.

genregdefs - каталог с исходныими текстами и базой данныз регистров процессоров для формирования CMSIS hedaders

rbf2c- сходный текст утилиты преобразования файлов двоисного образа инициализации FPGA в текст на C

dfu-util-0.6-win32.zip и dfu-util-0.6-win64.zip
Взятые с dfu-util.sourceforge.net утилиты командной строки для обноаления прошивки устройсв по протоколу DFU.
