Storch_HF_TRX_CAT.inf
установочный файл ("драйвер") последовательного USB порта для windows XP, Windows7..8.1. Windows 10 уже не требует использования этого файла.

convert_rbf_to_c.cmd
Командный файл преобразования файлов двоисного образа инициализации FPGA в текст на C. Использует windows 32bit утилиту rbf2c.exe.

rbf2c.c
Исходный текст утилиты преобразования файлов двоисного образа инициализации FPGA в текст на C. При установленном компиляторе может быть скомпилирована например такой командной строкой:
cl rbf2c.c
