Каталог для сборки версии программы под процессор XILINX XC7Z010/XC7Z015/XC7Z020

Программа:
Выполнить make clean, затем make
Результатом работы является файл tc1_xc7z020_app.stm32

Загрузчик:
Выполнить make clean, затем make bootloader
Результатом работы является файл tc1_xc7z020_boot.hex и tc1_xc7z020_boot.stm32
Требуются изменения в product.h
