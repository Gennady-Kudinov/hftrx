Каталог для сборки версии программы под процессор XILINX Zynq UltraScale+ Device XCZU2..XCZU9, XCZU11

Программа:
Выполнить make clean, затем make
Результатом работы является файл tc1_xc7z010_app.stm32

Загрузчик:
Выполнить make clean, затем make bootloader
Результатом работы является файл tc1_xc7z010_boot.hex и tc1_xc7z010_boot.stm32
Требуются изменения в product.h
